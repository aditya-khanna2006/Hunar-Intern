# 📘 Assignment 1: NumPy & SQL Basics

## 📌 Overview

This assignment was completed as part of the **Data Science Internship at HUNAR INTERN**.

The objective of this assignment was to strengthen the fundamentals of **Python (NumPy)** and **SQL** by implementing matrix operations and database queries.

---

## 🎯 Objectives

### Task 1 – NumPy

- Create a square matrix using user input
- Convert the input into a NumPy array
- Calculate the sum of diagonal elements using a loop
- Display the matrix and the diagonal sum

### Task 2 – SQL

- Create Student and Course tables
- Insert sample records
- Perform SQL queries to retrieve meaningful information
- Understand the use of primary key and foreign key
