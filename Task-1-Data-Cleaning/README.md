# 🧹 Task 1: Data Cleaning

## 📌 Overview

This project was completed as part of the **Data Science Internship at HUNAR INTERN**.

The objective of this task was to clean and preprocess a raw dataset using Python and Pandas. The project focuses on improving data quality by handling missing values, removing duplicate records, and preparing the dataset for further analysis.

---

## 🎯 Objectives

- Import and inspect the dataset
- Identify missing values
- Handle missing data using appropriate techniques
- Remove duplicate records
- Standardize and clean the dataset
- Export the cleaned dataset

## 📋 Data Cleaning Process

The following preprocessing steps were performed:

- Explored the dataset structure
- Identified missing values
- Filled missing values where appropriate
- Removed duplicate records
- Verified data consistency
- Exported the cleaned dataset
