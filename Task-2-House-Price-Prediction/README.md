# 🏠 Task 2: House Price Prediction using Linear Regression

## 📌 Overview
This project was completed as part of the **Data Science Internship at HUNAR INTERN**.

The objective of this task was to build a Machine Learning model using **Linear Regression** to predict house prices. The project includes data preprocessing, feature engineering, model training, model evaluation, and prediction using Python and Scikit-learn.

---

## 🎯 Objectives
- Import and inspect the dataset
- Perform data preprocessing
- Convert categorical data into numerical format
- Split the dataset into training and testing sets
- Train a Linear Regression model
- Evaluate model performance
- Predict house prices
- Visualize model results

## 📋 Machine Learning Workflow

The following steps were performed:

- Loaded the dataset
- Explored and understood the dataset
- Checked missing values and duplicate records
- Removed unnecessary columns
- Converted categorical variables into numerical values
- Split the dataset into training and testing sets
- Trained the Linear Regression model
- Evaluated the model using different performance metrics
- Predicted house prices
- Visualized the prediction results

---

## 📊 Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
